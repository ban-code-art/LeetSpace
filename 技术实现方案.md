# LeetSpace 技术实现方案

## 技术选型

| 维度 | 选择 | 说明 |
|------|------|------|
| 框架 | React 18 + TypeScript | 组件化开发，生态丰富 |
| 构建 | Vite + CRXJS | 专为 Chrome 插件优化的构建工具 |
| UI 库 | Tailwind CSS + shadcn/ui | 轻量、可定制、无运行时开销 |
| 存储 | chrome.storage.local | 纯本地，隐私优先 |
| 状态管理 | Zustand | 轻量，适合插件场景 |
| 插件规范 | Manifest V3 | Chrome 最新标准 |
| AI 调用 | 用户自带 API Key，前端直调 | 无后端成本 |

## 核心数据模型

```typescript
// 题目信息（从力扣页面提取）
interface Problem {
  id: string;              // 力扣题号
  title: string;           // 题目标题
  titleSlug: string;       // URL slug
  difficulty: 'Easy' | 'Medium' | 'Hard';
  tags: string[];          // 算法标签
  url: string;             // 题目链接
}

// 计划条目
interface PlanItem {
  id: string;              // UUID
  problem: Problem;
  status: 'todo' | 'done';
  date: string;            // 计划日期 YYYY-MM-DD
  createdAt: number;
  completedAt?: number;
}

// 笔记
interface Note {
  id: string;              // UUID
  problemId: string;       // 关联题目
  problem: Problem;
  content: string;         // Markdown 内容
  approach: string;        // 解题思路摘要
  tags: string[];          // 自定义标签（如：动态规划、双指针）
  difficulty: 1 | 2 | 3 | 4 | 5;  // 个人难度评分
  createdAt: number;
  updatedAt: number;
}

// 设置
interface Settings {
  dailyGoal: number;       // 每日目标题数
  theme: 'light' | 'dark' | 'system';
  ai: {
    provider: 'openai' | 'claude' | 'deepseek';
    apiKey: string;
    model: string;
    baseUrl?: string;      // 自定义 API 地址
  };
}
```

## 模块实现细节

### 1. Side Panel 机制

Chrome Side Panel API 允许插件在浏览器右侧打开一个持久面板。

**manifest.json 关键配置：**

```json
{
  "manifest_version": 3,
  "name": "LeetSpace",
  "version": "0.1.0",
  "permissions": [
    "sidePanel",
    "storage",
    "activeTab",
    "tabs"
  ],
  "side_panel": {
    "default_path": "src/sidepanel/index.html"
  },
  "background": {
    "service_worker": "src/background/index.ts"
  },
  "content_scripts": [
    {
      "matches": ["*://leetcode.cn/*", "*://leetcode.com/*"],
      "js": ["src/content/index.ts"]
    }
  ],
  "action": {
    "default_icon": "icons/icon-48.png"
  }
}
```

**打开方式：** 点击插件图标时打开 Side Panel（而非 Popup）。

```typescript
// background/index.ts
chrome.action.onClicked.addListener((tab) => {
  chrome.sidePanel.open({ tabId: tab.id });
});
```

### 2. 刷题计划系统

**核心交互流程：**

1. 用户在力扣题目页面时，Content Script 提取题目信息
2. Side Panel 显示「添加到今日计划」按钮
3. 用户勾选完成后，自动记录完成时间
4. 支持按日期查看历史计划和完成率

**Content Script 提取题目信息：**

```typescript
// content/index.ts
function extractProblemInfo(): Problem | null {
  // leetcode.cn 的题目页 URL 格式: /problems/{slug}/
  const match = location.pathname.match(/\/problems\/([\w-]+)/);
  if (!match) return null;

  const slug = match[1];
  const titleEl = document.querySelector('[data-cy="question-title"]')
    || document.querySelector('.text-title-large');
  const diffEl = document.querySelector('[diff]')
    || document.querySelector('.text-difficulty-easy, .text-difficulty-medium, .text-difficulty-hard');

  return {
    id: '', // 从页面或 API 获取
    title: titleEl?.textContent?.trim() || slug,
    titleSlug: slug,
    difficulty: parseDifficulty(diffEl),
    tags: extractTags(),
    url: location.href,
  };
}

// 监听来自 Side Panel 的消息
chrome.runtime.onMessage.addListener((msg, sender, sendResponse) => {
  if (msg.type === 'GET_PROBLEM_INFO') {
    sendResponse(extractProblemInfo());
  }
});
```

### 4. AI 辅助

**定位：** 轻量辅助，不替用户思考。

**功能场景：**

| 场景 | Prompt 策略 |
|------|-------------|
| 思路提示 | 给出解题方向，不给代码 |
| 复杂度分析 | 分析用户代码的时间/空间复杂度 |
| 解法对比 | 对比不同解法的优劣 |
| 笔记润色 | 帮助整理笔记结构 |

**实现：**

```typescript
// services/ai.ts
async function chat(messages: Message[], settings: Settings['ai']): Promise<string> {
  const { provider, apiKey, model, baseUrl } = settings;

  const url = baseUrl || getDefaultBaseUrl(provider);

  const response = await fetch(`${url}/v1/chat/completions`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${apiKey}`,
    },
    body: JSON.stringify({ model, messages }),
  });

  const data = await response.json();
  return data.choices[0].message.content;
}
```

**安全：** API Key 存储在 chrome.storage.local，不会上传到任何第三方服务。
